
Git Repositories
----------------

* UI and Django: git@github.com:benthomasson/fsm-designer-svg.git

Push to main repo:

    $ git push


Push to UI repo:

    $ git subtree push --prefix prototype/static git@github.com:benthomasson/svg-test.git master
